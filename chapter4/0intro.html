引言¶

本章导读¶

本章中内核将实现虚拟内存机制，这注定是一趟艰难的旅程。

实践体验¶

本章应用运行起来效果与上一章基本一致。

获取本章代码：

$ git clone https://github.com/LearningOS/rust-based-os-comp2022.git
$ cd rust-based-os-comp2022/

在 qemu 模拟器上运行 lab2(os4)参考框架: ：

$ cd os4-ref
$ make run
